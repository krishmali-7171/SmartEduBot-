# 🤖 SmartEduBot

SmartEduBot is an AI-powered college and placement assistance chatbot designed to help students with:

- 📘 DSA (Data Structures & Algorithms)
- 🧠 Aptitude preparation
- 💼 Mock interview practice
- 🎯 Placement guidance

Built using Node.js, Express.js, JavaScript, JWT Authentication, and OpenAI API.

---

✅ AI Chatbot Assistance

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/krishmali-7171/SmartEduBot-.git
```

---

## 2️⃣ Open Project

```bash
cd SmartEduBot-
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Create Environment Variables

Create a `.env` file in the root directory and add:

```env
OPENAI_API_KEY=your_openai_api_key
JWT_SECRET=your_secret_key
```

---

## 5️⃣ Run the Server

```bash
npm start
```
